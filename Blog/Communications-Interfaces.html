Communications & Interfaces

This blog entry explores all of the computer hardware and sofware I prefer and use on a daily basis. Most software is command-line-based and for the pragmatic man only. I made the switch to the Linux distro ‘Manjaro’ in the latter end of 2020, using a Tiling Window Manager. I’ve also created a similar setup on my Mac which allows a more non-MacOS method of communicating with MacOS. This is simply a far more effcient and easier way to operate machines. Specifically, with regards to the Mac, the best thing I’ve ever done is hide that goddamn dock and menu bar!

Usual Workflow

I am developing and maintaining a sustainable and minimalist Linux system. I main a Lenovo ThinkPad T420 and burn an X220. I currently use the OS Manjaro on which I run i3-Gaps but with plans to transition to Suckless’ dwm window tiling manager. I tend to spend my days in the text editor Vim, writing and compiling documents in LaTeX/Markdown using primarily pandoc and pdflatex. This sometimes extends to Beamer-compiled slide shows. I’m also very familiar with my desktop workhorse and other machines running Windows XP and Windows 7 and, my MacBook Pro running MacOS.
